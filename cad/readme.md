Step, STL, and Alibre design files for the case, and DXF files for the switchplate.
The top case uses eight M3 heat-set inserts.  Then you need six M3x5 standoffs and eight M3x7 screws.
Even then, there is some slight jank involved. Proceed at your own risk.  The "hybrid-wired" PCB is documented in the "N3V4" repository.
